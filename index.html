Introduction to Docker Volume Driver for vSphere¶

Docker volume driver is designed to solve persistency needs for stateful containers running on top of VMware’s stack.

Some of the high level features for the plugin are:

  1. Designed to run over shared storage in a cluster (Single node setup for testing is supported)
  2. Easy to deploy and manage. There is zero configuration and zero credential management.
  3. Support for VSAN policy.
  4. Integration with vCenter (under development)
  5. Backup of Docker volumes (under development)

This plugin is integrated with Docker Volume Plugin framework. This plugin does not need credential management or configuration management.
